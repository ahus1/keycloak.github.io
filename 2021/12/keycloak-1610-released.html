Keycloak 16.1.0 released

December 20 2021

To download the release go to Keycloak downloads.

Release notes

Highlights

Upgrade to Wildfly 26.0.0

Keycloak server was upgraded to use Wildfly 26.0.0.Final as the underlying container.

For more information on WildFly 26 refer to the WildFly 26 release notes.

Migration from 16.0

Before you upgrade remember to backup your database. If you are not on the previous release refer to the documentation for a complete list of migration changes.

All resolved issues

Enhancements

  • #9219 Upgrade to WildFly 26.0.0.Final keycloak

Upgrading

Before you upgrade remember to backup your database and check the upgrade guide for anything that may have changed.
